Load candidates from the JSON file. For each candidate, render a Leaflet circleMarker at their lat/lng coordinates. Color-code markers by office type (e.g., state house = blue, state senate = green, county = orange, city council = purple, school board = teal). Define the color mapping in a config object. Add a simple legend overlay showing the office type colors.
